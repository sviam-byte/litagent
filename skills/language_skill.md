- LANGUAGE_PATHOLOGIES
- REGISTER_AND_TONE
- METAPHOR_AND_ABSTRACTION
- RHYTHM_AND_SOUND

## Inputs
- raw_text
- corpus_profile
- segment_map

## Output
- language pathology clusters
- tone/register issues
- metaphor/abstraction issues
