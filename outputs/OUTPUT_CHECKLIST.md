# OUTPUT_CHECKLIST

Перед отдачей финального результата проверь:
- есть ли corpus limits
- есть ли final diagnosis
- есть ли branch-specific analysis
- есть ли strengths
- есть ли false positive cautions
- есть ли priority buckets
- есть ли revision plan
- нет ли дублей между language/sentence/paragraph/macro
