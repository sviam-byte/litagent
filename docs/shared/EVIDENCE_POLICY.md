# EVIDENCE_POLICY

## Rule
Сильное замечание должно иметь текстовую опору.

## Evidence Types
- single decisive span
- repeated pattern across multiple spans
- structural pattern visible across sections

## Forbidden
- громкий вывод по одному слабому примеру
- диагноз уровня “весь текст” без репрезентативных span-ов
