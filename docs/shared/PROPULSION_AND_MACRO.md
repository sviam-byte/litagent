# PROPULSION_AND_MACRO

## Goal
Понять, за счёт чего текст удерживает внимание и как распределяет вес.

## Questions
- Почему читатель идёт дальше?
- Какие вопросы текст открывает?
- Где они выплачиваются?
- Где текст провисает?
- Соответствует ли объём важности?
- Не повторяет ли текст один и тот же жест?

## Common Failures
- тема есть, движения нет
- локальные крючки без выплаты
- раздутый вход
- мёртвая середина
- скомканный финал
- несоразмерный вес второстепенному
